# 📚 Books Data Scraper

PythonとBeautifulSoupを使用して、テスト用スクレイピングサイト（http://books.toscrape.com/）から全50ページ（1000件）の書籍データを自動取得し、CSVファイルとして保存するプログラムです。

## 🎯 取得するデータ

このプログラムは、各書籍から以下の情報を抽出します。

* タイトル (Title)
* 価格 (Price)
* 評価 (Rating)
* 在庫状況 (Stock) ※「在庫あり」「在庫なし」等、日本語に自動翻訳して取得します
* 商品URL (URL)

取得したデータは `data` フォルダ内に、実行日時のついたCSVファイル（例: `20260310_books.csv`）として保存されます。

## ✨ 主な機能

- **全ページ自動巡回**: ページネーション（次へボタン）を自動で判別し、最終ページまで取得を続けます。
- **データの日本語化**: 英語のステータスを分かりやすい日本語に変換して保存します。
- **エラーハンドリング**: 途中のページや一部のデータでエラーが起きても、プログラムがクラッシュせずに次の処理へ進む安全設計です。
- **モジュール分割設計**: 処理の「司令塔」「スクレイピング職人」「保存職人」を分割し、保守性を高めています。

## 🛠️ 使用技術

* Python 3.x
* requests (HTTP通信)
* BeautifulSoup4 (HTMLパース・スクレイピング)
* csv (標準ライブラリ)

## 🏗️ フォルダ構成

```text
books_data_scraper/
│
├── main_flow.py          # 司令塔：全体の処理フロー（スクレイピング〜保存）を管理
├── config.py             # 設定ファイル：ターゲットURLやCSVの日本語ヘッダー情報を一元管理
├── requirements.txt      # 必要な外部ライブラリのリスト
├── README.md             # このファイル
│
├── data/                 # 取得したCSVファイルが保存されるディレクトリ
│
├── scraper/
│   └── books_scraper.py  # 抽出職人：HTMLの取得、ページ遷移、データのパース
│
└── utils/
    ├── save_data.py      # 保存職人：取得したデータをCSVに書き込む機能
    ├── logger.py         # ログ出力用モジュール
    └── path_helper.py    # パス管理用モジュール

```

## 💻 セットアップ方法

### 1. 仮想環境を作成

```bash
python3 -m venv venv

```

### 2. 仮想環境を有効化

**Mac / Linux の場合**

```bash
source venv/bin/activate

```

**Windows の場合**

```bash
venv\Scripts\activate

```

### 3. ライブラリをインストール

```bash
pip install -r requirements.txt

```

## 🚀 実行方法

ターミナルで以下のコマンドを実行します。

```bash
python main_flow.py

```

実行するとターミナルに進行状況のログが表示され、全ページのデータ取得完了後、`data` フォルダ内にCSVファイルが保存されます。

## ⚠️ 参考サイト・注意事項

* **対象サイト**: http://books.toscrape.com/
* ※ このサイトはスクレイピング練習用に公開されているテストサイトです。
* 本プログラムを他の実際のECサイトに転用する場合は、必ず対象サイトの利用規約や `robots.txt` を確認し、サーバーに負荷をかけないよう適切なアクセス間隔（スリープ処理）を設けてください。
